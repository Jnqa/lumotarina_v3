# Rewind Games Format

## S3 структура

```
/rewind/
  rewind.json                    ← Список всех игр
  games/
    game-2026-01-15.md          ← Содержимое игры (markdown)
    game-2026-03-20.md
    ...
```

## Формат `/rewind/rewind.json`

```json
{
  "games": [
    {
      "id": "game-2026-02-21",
      "title": "Красная Пирамида",
      "date": "2026-02-21",
      "description": "Партия начинается с встречи персонажей в таверне. Они узнают о загадочном артефакте и получают первое задание.",
      "characters": [
        "Арагон (воин)",
        "Мирия (маг)",
        "Локи (вор)"
      ],
      "charactersID": [],
      "content": "/rewind/games/game-2026-02-21.md",
      "public": false,
      "allowedUserIds": []
    },
    {
      "id": "game-2026-03-20-private",
      "title": "Закрытая сессия - Тайна древних",
      "date": "2026-03-20",
      "description": "Секретная встреча с членами Совета. Только для участников этой сессии.",
      "characters": [
        "Саша (маг)",
        "Артем (клирик)"
      ],
      "charactersID": [],
      "content": "/rewind/games/game-2026-03-20.md",
      "public": false,
      "allowedUserIds": [
        "user-123",
        "user-456"
      ]
    }
  ]
}
```

## Поля игры

| Поле | Тип | Обязательное | Описание |
|------|-----|--------------|---------|
| `id` | string | ✅ | Уникальный ID игры |
| `title` | string | ✅ | Название сессии |
| `date` | string | ✅ | Дата проведения (YYYY-MM-DD) |
| `description` | string | ✅ | Краткое описание того, что произошло |
| `characters` | string[] | ✅ | Массив имен персонажей участников |
| `content` | string | ✅ | Путь к .md файлу с полным рассказом |
| `public` | boolean | ✅ | Публична ли игра (видна всем) |
| `allowedUserIds` | string[] | ✅ | ID пользователей с доступом (если не public) |
| `allowedTgIds` | string[] | ✅ | Telegram ID с доступом (если не public) |

## Логика доступа (фронт)

```typescript
const canView = item.public || 
  (tgId && item.allowedTgIds.includes(tgId)) || 
  (userId && item.allowedUserIds.includes(userId));
```

Игра видна пользователю если:
1. `public: true` — видна всем
2. `public: false` + Telegram ID пользователя в `allowedTgIds` — видна только этому пользователю
3. `public: false` + User ID пользователя в `allowedUserIds` — видна только этому пользователю

## Пример .md файла контента

```markdown
# Первая сессия - Знакомство

**Дата:** 15 января 2026  
**Партия длилась:** 3 часа 45 минут

## Персонажи
- Арагон (воин)
- Мирия (маг)
- Локи (вор)

## Краткое резюме
Партия начинается в таверне "Веселый гном". Наши герои встречаются впервые...

## Ключевые события
1. Первая встреча в таверне
2. Получение квеста от Старого странника
3. Начало расследования в замке...

## Результаты
- Найден артефакт
- Раскрыта тайна замка
- Персонажи получили 500 опыта

## Цитаты игроков
> "Я делаю что-то безумное" — Локи перед прыжком в пропасть

> "Давай бросим монету" — Мирия перед сложным решением
```

## Миграция из старого формата

Если у вас был `/rewind_list.json`:

```json
[
  {
    "id": "session-2026-01-10",
    "title": "Сессия 10 января 2026",
    "description": "...",
    "url": "https://...",
    "public": true
  }
]
```

Преобразуйте в новый формат:

```json
{
  "games": [
    {
      "id": "game-2026-01-10",
      "title": "Сессия 10 января 2026",
      "date": "2026-01-10",
      "description": "...",
      "characters": [],
      "content": "https://...",
      "public": true,
      "allowedUserIds": [],
      "allowedTgIds": []
    }
  ]
}
```
